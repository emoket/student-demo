# student-demo
vue + springboot + mariadb + crud + elk

# how to use

## frontend (vue)

```sh
cd vue_crud_demo
npm install & npm start
```

## backend (springboot)

```sh
cd springboot_crud_demo
# java run!
```

## database (mariadb)

```sh
cd mariadb
docker-compose up -d
# .env 파일의 값을 원하는대로 변경해도 무방
```

## elk

```sh
cd docker-elk
docker-compose up -d
```
